SpeechSynthesisUtterance

• The SpeechSynthesisUtterance interface of the Web Speech API represents a speech request. It contains the content the speech service should read and information about how to read it (example language, pitch and volume).

• Returns a new SpeechSynthesisUtterance object instance.

• SpeechSynthesisUtterance also inherits from its parent interface, EventTarget.

• SpeechSynthesisUtterance.lang - Gets and sets the language of the utterance.

• SpeechSynthesisUtterance.pitch- Gets and sets the pitch at which the utterance will be spoken at.

• SpeechSynthesisUtterance.rate - Gets and sets the speed at which the utterance will be spoken at.

• Listen to these events using addEventListener or by assigning an event listener to the oneventname property of this interface.

voiceChanged event

• The voiceChanged event of the Web Speech API is fired when the list of SpeechSynthesisVoice objects that would be returned by the SpeechSynthesis.

• The voicechanged event of the Web Speech API is fired when the list of SpeechSynthesisVoice objects that would be returned by the SpeechSynthesisgetVoices() method has changed (when the voicechanged event fires).

• Use the event name in methods like addEventListener(), or set an event handler property.

• SYNTAX
addEventListener("voicechanged", event => {})
onvoicechanged = event => {}

JavaScript Array find()

• The find() method returns the value of the first element that passes a test.

• The find() method executes a function for each array element.

• The find() method returns undefined if no elements are found.

• The find() method does not execute the function fo empty elements.

• The find() method does not change the original array.

• The find() method is ECMAScript(ES6) feature.

• SYNTAX
array.find(function(currentValue, index, arr), thisValue)
